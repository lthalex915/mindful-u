AI Configuration

Configure your AI provider (OpenRouter, OpenAI Compatible, or Deepseek), API key, and model. Your credentials are stored locally and never sent to our servers.

Choose which API you want to use. "OpenAI Compatible" supports APIs that follow the OpenAI Chat Completions spec (e.g., OpenAI, local gateways, etc.).

For OpenAI-compatible APIs, enter the base URL (e.g., https://api.openai.com/v1). For Deepseek, default is https://api.deepseek.com/v1.

Get your API key from OpenRouter

Enter the model identifier supported by your selected provider.

Examples:
• OpenRouter: deepseek/deepseek-chat-v3-0324:free
• OpenRouter: minimax/minimax-m2:free
• Deepseek: deepseek-chat
• OpenAI Compatible: gpt-4o-mini

Paste a JSON config. Supported providers: "openrouter", "openai_compatible", "deepseek".
